# AstroPenguim
a page just to acquire knowledge, using Vite and the React + Typescript template

```bash
# use to install project dependencies
$ yarn install 

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start

```
